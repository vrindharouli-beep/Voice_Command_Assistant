# Voice_Command_Assistant
A Python-based smart desktop voice assistant with a CustomTkinter GUI that integrates speech recognition, text-to-speech, system automation, and Google Gemini API for intelligent conversational responses.



---

##  Features

- Real-time voice recognition
- Text-to-speech responses
- Opens websites and applications
- Provides date and time information
- Retrieves system information
- AI-powered responses using Gemini API
- Modern GUI built with CustomTkinter

---

##  Technologies Used

- Python
- CustomTkinter
- SpeechRecognition
- pyttsx3
- Google Gemini API
- psutil
- NumPy
- SoundDevice
- dotenv
